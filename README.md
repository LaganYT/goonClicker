# Goon Clicker - React Native Idle Game

A feature-rich idle clicker game built with React Native and Expo, inspired by Cookie Clicker.

## 🎮 Features

### Core Gameplay
- **Click to Earn**: Tap the goon button to earn goons
- **Auto-Clickers**: Purchase upgrades to automatically generate goons
- **Prestige System**: Reset progress for permanent multipliers
- **Achievements**: Unlock achievements for rewards and progression
- **Daily Rewards**: Claim daily rewards for bonus goons
- **Special Events**: Participate in time-limited events

### Audio System 🎵
- **Sound Effects**: Click sounds, upgrade sounds, achievement sounds
- **Background Music**: Ambient background music with volume control
- **Audio Settings**: Toggle sound effects and music independently
- **Volume Control**: Adjustable volume sliders for both sound and music

### Visual Effects ✨
- **Particle Effects**: Animated particles when clicking (toggleable)
- **Floating Text**: Numbers float up when earning goons
- **Smooth Animations**: Button press animations and transitions
- **Theme Support**: Dark, Light, and Neon themes

### Statistics & Analytics 📊
- **Detailed Statistics**: Track play time, clicks, efficiency, and more
- **Performance Metrics**: Click rate, goon efficiency, fastest prestige
- **Session Tracking**: Monitor gaming sessions and progress
- **Historical Data**: View highest achievements and milestones

### Notification System 🔔
- **In-Game Notifications**: Real-time notifications for achievements, upgrades, and events
- **Notification Center**: Dedicated screen to view and manage notifications
- **Smart Notifications**: Context-aware notifications with action buttons
- **Notification Badge**: Unread notification counter on the main screen

### Settings & Customization ⚙️
- **Audio Settings**: Sound effects, music, and volume controls
- **Visual Settings**: Particle effects and theme selection
- **Game Settings**: Auto-save, notifications, and vibration
- **Prestige Management**: View prestige progress and requirements

### Enhanced Gameplay Features 🚀
- **Offline Progress**: Earn goons while the app is closed
- **Auto-Save**: Automatic game state saving every 30 seconds
- **Haptic Feedback**: Vibration feedback for interactions (where supported)
- **Responsive Design**: Optimized for different screen sizes

## 🔧 Installation & Setup

1. **Prerequisites**
   ```bash
   npm install -g expo-cli
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm start
   ```

4. **Run on Device/Simulator**
   ```bash
   # iOS
   npm run ios
   
   # Android
   npm run android
   
   # Web
   npm run web
   ```
